# Neopets Sales History Enhancer

A lightweight Tampermonkey userscript that enhances the Neopets **Sales History** page with powerful sorting, searching and grouping features while preserving the original Neopets look and feel.

## Features

- 🔎 Search sales by item or buyer
- 📦 Group duplicate item sales into a single summary row
- ↕️ Sort every column by clicking the table headers
- 📊 View total sales and sales statistics
- 🔄 Instantly switch between individual and grouped views
- 🎨 Designed to blend seamlessly with the classic Neopets interface

## Installation

### 1. Install Tampermonkey

Download the Tampermonkey browser extension:

- Chrome
- Firefox
- Edge
- Safari

### 2. Install the Script

Create a new userscript and paste in the contents of `Neopets Sales History Enhancer.user.js`.

Save the script.

### 3. Visit Your Sales History

Navigate to:

```
https://www.neopets.com/market.phtml?type=sales
```

The enhancements will be applied automatically.

## Usage

### Sorting

Click any table header to sort.

- First click → Ascending
- Second click → Descending

### Group Items

Click **📦 Group Items** to combine duplicate sales into a single row showing:

- Quantity sold
- Lowest price
- Average price
- Highest price
- Total value

Click **📋 Individual Items** to return to the original view.

### Search

Use the search box to quickly filter sales.

## Why?

Neopets' sales history is great for checking recent sales, but becomes difficult to use once you've sold hundreds or thousands of items.

This userscript adds quality-of-life improvements while keeping the original page familiar and lightweight.


## Compatibility

Tested with:

- Google Chrome
- Tampermonkey

Should also work with most Chromium-based browsers.
